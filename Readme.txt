node(express), angular, postgresql

Database dump
        - dump -> ./dump.sql
        
initialisation
    Client
        1 - cd ./Client
        2 - npm install

    Server
        1 - cd ./Server
        2 - npm install
        
Run
    Client
        1 - cd ./Client
        2 - ng serve --port 8001

    Server
        1 - cd ./Server
        2 - npm run dev (or npm run start)

Open url in the browser
    http://localhost:8001
